## Local Development

This application requires Node.js v18+.

```bash
git clone https://github.com/nurodev/nuro.dev.git
cd nuro.dev
yarn install
yarn dev
```

Create a .env file similar to [.env.example](https://github.com/nurodev/nuro.dev/blob/main/.example.env).
